# DERRR hosted asset pack

These files are the canonical public-hosted DERRR brand assets for reuse across repos, docs, and social surfaces.

Expected public URLs:

- `/assets/derrr/banner.svg`
- `/assets/derrr/logo-primary.svg`
- `/assets/derrr/icon.svg`
- `/assets/derrr/logo-mono.svg`
- `/assets/derrr/loop-diagram.svg`

Source of truth for editable masters currently lives in the DERRR repo:
- `Zzzero-hash/derrr`
- `assets/`

When updating this pack:
1. edit the source asset in the DERRR repo
2. copy/export the canonical hosted version here
3. deploy the site so `grovextech.com` serves the updated asset

Note: social preview PNG can be added here later once exported from the SVG source.
